# Failed Approaches Log

## Browser-Based Prospecting
**When:** 2026-03-21
**What:** Tried to browse Instagram/TikTok profiles directly
**Why it failed:** Login walls, reCAPTCHA blocks
**Resolution:** Switched to kimi_search + website extraction
**Lesson:** Search beats browser for low-budget scraping

## Gmail for Agent Email
**When:** 2026-03-20
**What:** Considered Gmail API for Scout inbox
**Why it failed:** Retrofitted for agents, no semantic search, hard to scale
**Resolution:** Selected AgentMail (purpose-built)
**Lesson:** Use tools designed for the job

## Single-File MEMORY.md
**When:** 2026-03-27 to 2026-03-30
**What:** All operational knowledge in one 20KB file
**Why it failed:** Context bloat, hard to navigate, mixed procedural with strategic
**Resolution:** 3-layer architecture (AGENTS.md → Skills → Deep Memory)
**Lesson:** Progressive disclosure, separate concerns
