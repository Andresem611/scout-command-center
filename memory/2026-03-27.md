### How It Works

1. **Every heartbeat run** (via cron):
   - Updates `scout_data.json['agent_status']` with timestamp and current task
   - Logs activity to activity_log (last 50 entries)

2. **Dashboard displays**:
   - 🟢 Green dot = seen <5 min ago
   - 🟡 Yellow dot = seen 5-30 min ago  
   - 🔴 Red dot = seen >30 min ago (likely down)
   - Current task shows what Scout is working on
   - Expandable "Recent Activity" section shows last 10 tasks

3. **Auto-refresh**:
   - Streamlit caches data for 10 seconds (`ttl=10`)
   - Page refreshes show new status within 10s of heartbeat
