# Pipeline Schema

## Data Structure

The pipeline data is stored in `scout_data.json` with the following structure:

```json
{
  "prospects": [
    {
      "id": "music-academy-nyc-001",
      "name": "Manhattan School of Music",
      "stage": "Negotiating",
      "city": "New York",
      "branch": "Conservatory",
      "contact_email": "partnerships@msm.edu",
      "contact_phone": "+1-212-555-0100",
      "source": "Google Maps",
      "notes": "Interested in piano referrals",
      "created_at": "2026-03-15T10:30:00+00:00",
      "updated_at": "2026-03-28T14:22:00+00:00",
      "days_in_stage": 5
    }
  ],
  "metadata": {
    "last_sync": "2026-03-30T03:00:00+00:00",
    "version": "1.0"
  }
}
```

## Required Fields

Every prospect record **must** have these fields:

| Field | Type | Description |
|:------|:-----|:------------|
| `id` | string | Unique identifier (alphanumeric, hyphens, underscores) |
| `name` | string | Business/organization name |
| `stage` | string | One of: Prospected, Contacted, Replied, Negotiating, Partner, Declined |
| `city` | string | City location |
| `branch` | string | Type of music business (Conservatory, Private Studio, School, etc.) |

## Optional Fields

| Field | Type | Description |
|:------|:-----|:------------|
| `contact_email` | string | Primary contact email |
| `contact_phone` | string | Primary contact phone |
| `source` | string | How prospect was discovered |
| `notes` | string | Free-form notes |
| `website` | string | Business website URL |
| `address` | string | Full street address |

## Computed Fields

These are auto-generated by the system:

| Field | Type | Description |
|:------|:-----|:------------|
| `created_at` | ISO datetime | When record was first created |
| `updated_at` | ISO datetime | Last modification timestamp |
| `days_in_stage` | integer | Days since last stage change |

## ID Format

- Use lowercase with hyphens: `city-business-name-001`
- Include city prefix for easy filtering
- Use sequential numbering for multiple locations
- Examples:
  - `nyc-julliard-001`
  - `la-hollywood-piano-studio-001`
  - `chicago-merit-school-001`

## Stage Values

Must be exactly one of:

| Value | Meaning |
|:------|:--------|
| `Prospected` | Initial discovery |
| `Contacted` | Outreach sent |
| `Replied` | Prospect responded |
| `Negotiating` | Active discussion |
| `Partner` | Signed up, active |
| `Declined` | Closed-lost |
