## Search Tips

1. **Combine terms** for better targeting:
   - "homeschool + mom + blog + {city}"
   - "{city} + parenting + instagram"

2. **Use quotes** for exact phrases in search engines:
   - `"Austin mom blogger"`

3. **Add platform** to narrow results:
   - Add "instagram" for social profiles
   - Add "blog" for websites
   - Add "tiktok" for video creators

4. **Exclude terms** with minus:
   - "mom blogger -celebrity -news"

5. **Look for these bio keywords:**
   - "Mom of [number]"
   - "Homeschool"
   - "SAHM" (stay at home mom)
   - "Lifestyle blogger"
   - "Parenting tips"
   - Link in bio tools (Linktree, Beacons)
