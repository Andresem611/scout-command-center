## Pattern Categories

### Reply Types
| Type | Definition | Response Strategy |
|------|------------|-------------------|
| INTERESTED | Wants to learn more / book call | Immediate alert, warm response, hand-off to Keri |
| ASKING QUESTIONS | Needs info before deciding | Detailed helpful reply, show for approval |
| ASKING ABOUT RATES | Wants pricing/partnership terms | Flag for Andres, don't commit without approval |
| DECLINED | Not interested | Polite thank you, mark cold |
| OUT OF OFFICE | Auto-reply | Keep follow-up schedule, no alert |
| SPAM | Irrelevant | Archive |
